Available MCP Tools
  • get_heap_summaryHeap overview: total size, object count, class count, JVM info
  • get_leak_suspectsMemory leak suspects with accumulation points and paths to GC roots
  • get_class_histogramTop classes by retained heap (filterable by class name regex)
  • get_dominator_treeTop objects dominating the most retained memory
  • get_top_consumersBiggest consumers grouped by class, classloader, and package
  • run_oql_queryExecute OQL (SQL-like) queries: SELECT * FROM java.lang.String WHERE @retainedHeapSize > 1000000
  • get_thread_stacksThread stack traces with local variables and retained sizes
